## 요청 흐름별 다이어그램

### 1. 정적 페이지 로딩

```
사용자 ──HTTPS──▶ Vercel CDN ──▶ 정적 파일 반환 (dist/)
                     │
                     │  캐시 HIT → 즉시 반환
                     │  캐시 MISS → Origin에서 가져온 후 캐싱
                     ▼
              React SPA 렌더링 (클라이언트)
```

### 2. AI 분석 요청 흐름

```
┌────────┐      ┌──────────┐      ┌───────────┐      ┌──────────┐
│ Browser │      │  Vercel  │      │  Google   │      │ Supabase │
│(React)  │      │Serverless│      │  Gemini   │      │   DB     │
└───┬────┘      └────┬─────┘      └─────┬─────┘      └────┬─────┘
    │                 │                   │                  │
    │ POST /api/analyze                   │                  │
    │ {resume, jobPosting}                │                  │
    │ + Authorization: Bearer <JWT>       │                  │
    ├────────────────▶│                   │                  │
    │                 │                   │                  │
    │                 │ generateContent() │                  │
    │                 │ (프롬프트 + 입력)   │                  │
    │                 ├──────────────────▶│                  │
    │                 │                   │                  │
    │                 │   JSON 분석 결과    │                  │
    │                 │◀──────────────────┤                  │
    │                 │                   │                  │
    │                 │ auth.getUser(JWT)                    │
    │                 ├────────────────────────────────────▶│
    │                 │                   │                  │
    │                 │              user 정보 반환           │
    │                 │◀────────────────────────────────────┤
    │                 │                   │                  │
    │                 │ INSERT INTO analyses                 │
    │                 ├────────────────────────────────────▶│
    │                 │                   │                  │
    │  200 OK         │                   │                  │
    │  {fitScore, ...}│                   │                  │
    │◀────────────────┤                   │                  │
    │                 │                   │                  │
```

### 3. 인증 흐름 (OAuth)

```
┌────────┐           ┌──────────┐           ┌───────────┐
│ Browser │           │ Supabase │           │  Google/  │
│(React)  │           │   Auth   │           │  Kakao    │
└───┬────┘           └────┬─────┘           └─────┬─────┘
    │                      │                       │
    │ signInWithOAuth()    │                       │
    ├─────────────────────▶│                       │
    │                      │                       │
    │  Redirect URL        │                       │
    │◀─────────────────────┤                       │
    │                      │                       │
    │──── 브라우저 리다이렉트 ─────────────────────▶│
    │                      │                       │
    │              OAuth 동의 화면                   │
    │◀─────────────────────────────────────────────┤
    │                      │                       │
    │──── 사용자 동의 ────────────────────────────▶│
    │                      │                       │
    │                      │  Authorization Code   │
    │                      │◀──────────────────────┤
    │                      │                       │
    │  Callback + JWT      │                       │
    │◀─────────────────────┤                       │
    │                      │                       │
    │  세션 저장 (localStorage)                     │
    │                      │                       │
```

### 4. 분석 이력 조회 흐름

```
┌────────┐           ┌──────────┐
│ Browser │           │ Supabase │
│(React)  │           │   DB     │
└───┬────┘           └────┬─────┘
    │                      │
    │ supabase.from('analyses')
    │ .select('*')
    │ .eq('user_id', uid)
    │ .order('created_at', desc)
    ├─────────────────────▶│
    │                      │
    │                      │ RLS 적용:
    │                      │ auth.uid() = user_id 확인
    │                      │
    │  본인 분석 이력만 반환  │
    │◀─────────────────────┤
    │                      │
```
